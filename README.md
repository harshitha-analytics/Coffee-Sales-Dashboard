# Coffee-Sales-Dashboard
Interactive Excel dashboard analyzing coffee sales trends and customer insights

## Project Overview
This project analyzes coffee sales data using Microsoft Excel and presents insights through an interactive dashboard.

## Features
- Interactive slicers and filters
- Sales trend analysis
- Country-wise sales comparison
- Top customer analysis
- Product category insights

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## Skills Demonstrated
- Data Visualization
- Dashboard Creation
- Business Analysis
- Data Cleaning
- Reporting
